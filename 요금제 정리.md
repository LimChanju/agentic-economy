# 2026년 경제 에이전트 API 요금제 정리

## 개요
agentic-economy 프로젝트에서 사용할 각 서비스의 요금제를 정리한 문서입니다.
초기에는 무료 요금제만 사용하며, 추후 유료 요금제로 전환 가능합니다.

---

## 암호화폐 API

| 서비스명 | 무료 플랜 | 유료 플랜 시작 | 비고 |
|---------|----------|--------------|------|
| **CoinGecko** | 월 10,000 크레딧 / 분당 30회 | 약 $35/월 (Basic) | 가장 방대한 코인 리스트 보유 |
| **CoinMarketCap** | 월 10,000 크레딧 / 과거 데이터 불가 | 약 $29/월 (Hobbyist) | 거래소 데이터에 강점 |
| **CryptoPanic** | 일 100회 요청 | 약 $29/월 (Pro) | 심리(Sentiment) 분석에 필수 |

### 추천 선택
- **기본 동향 파악**: CoinGecko (코인 수, 기본 정보가 가장 풍부)
- **거래 데이터**: CoinMarketCap (거래소 정보, 거래량)
- **심리 분석**: CryptoPanic (시장 센티먼트, 뉴스)

---

## 주식/경제 API

| 서비스명 | 무료 플랜 | 유료 플랜 시작 | 비고 |
|---------|----------|--------------|------|
| **Alpha Vantage** | 일 25회 (제한적) | 약 $50/월 (75 req/min) | 금리, GDP 등 거시 지표 제공 |
| **Polygon.io** | 당일 데이터 제외 (지연 데이터) | 약 $29/월 (Starter) | 미국 주식 실시간 데이터 강점 |
| **yfinance (Lib)** | 완전 무료 | - | API 키 없이 Yahoo Finance 크롤링 |

### 추천 선택
- **기본 주식 데이터**: yfinance (완전 무료, 추가 설정 불필요)
- **거시 경제 지표**: Alpha Vantage (금리, GDP, 실업률 등)
- **실시간 데이터**: Polygon.io (유료 전환 시)

## 뉴스/검색 API

| 서비스명 | 무료 플랜 | 유료 플랜 시작 | 비고 |
|---------|----------|--------------|------|
| **Tavily Search** | 월 1,000회 검색 | 약 $30/월 (4,000회) | AI 에이전트 전용 검색 엔진 |
| **NewsAPI** | 일 100회 (개발용) | 약 $40/월 (Basic) | 글로벌 언론사 뉴스 수집 |

### 추천 선택
- **에이전트 검색**: Tavily Search (AI 친화적)
- **뉴스 수집**: NewsAPI (언론사 기반)

## 소셜 미디어 API

| 서비스명 | 무료 플랜 | 유료 플랜 시작 | 비고 |
|---------|----------|--------------|------|
| **X (Twitter)** | 거의 쓰기 전용 (읽기 제한) | 약 $200/월 (Basic) | 가격이 가장 비싸고 진입장벽 높음 |

### 추천 선택
- **초기 단계**: Tavily Search로 대체 (X 포스트 검색 포함)
- **X 직접 API**: 필요시 유료 전환 고려

## 초기 구현 전략 (무료 요금제 기반)

### Phase 1: 데이터 수집
- ✅ CoinGecko (암호화폐)
- ✅ yfinance (주식)

- ✅ Tavily Search (뉴스/X 검색)

### Phase 2: 유료 전환
- CoinMarketCap (거래소 데이터)
- CryptoPanic (심리 분석)
- NewsAPI (전문 뉴스)
- Polygon.io (실시간 주식)

### Phase 3: 고급 기능
- X API Basic (주요 인물 팔로잉)
